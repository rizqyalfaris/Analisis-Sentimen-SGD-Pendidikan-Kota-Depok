Tentang Website

Website ini merupakan website untuk analisis sentimen pengguna Twitter terhadap Pendidikan Kota Depok. Tujuannya yaitu membantu kemendikbud, pemerintah Kota Depok ataupun pengguna lain dalam menganalisis kepuasan masyarakat melalui opini di Twitter sebagai bahan evaluasi pengelolaan pendidikan seperti kurikulum, sistem pembelajaran, prestasi dan lain-lainnya.
Website ini melakukan analisis dengan model Stochastic Gradient Descent (SGD). Model pembelajaran mesin ini sudah dilatih dan siap untuk digunakan pada data yang diunggah pengguna website. Website ini dibangun menggunakan Framework Flask.
